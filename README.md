<h1>GetX Todo</h1>

</br>To Do App created in Flutter

</br>Features:
</br>• GetX Package
</br>• GetX Storage
</br>• Firebase Auth

</br>How to use?
</br>1. Create a To Do list by entering a text and pressing the + icon.
</br>2. Edit an item by tapping the pencil icon.
</br>3. Delete an item by swiping left/right.
